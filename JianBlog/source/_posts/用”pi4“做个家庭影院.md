## 准备

您好！用树莓派 Pi 4B 搭建一个高清（4K）媒体播放器（通常运行 **Kodi** 等软件，作为 HTPC）是一个非常流行的应用。

Pi 4B 拥有强大的 4Kp60 HEVC 硬件解码能力和双 4K 输出支持，非常适合做媒体中心。

## 🖥️ 搭建媒体中心的功能与性能分析

### 1. 核心软件

最流行的选择是：

* **LibreELEC / OSMC：** 这是基于 Linux 的精简系统，专门为运行 **Kodi**（一个强大的开源媒体中心软件）而优化。它们启动快，性能优秀，是媒体播放的首选。
* **Raspberry Pi OS + Kodi：** 如果你还需要树莓派同时做其他事情（比如运行后台服务器），可以选择在完整的 Pi OS 上安装 Kodi。

### 2. Pi 4B 的优势（为何选它？）

| 特性 | 性能表现 | 对媒体播放的意义 |
| :---: | :---: | :---: |
| **4K 解码** | 硬件支持 **4Kp60 HEVC** 解码。 | 流畅播放高码率的 4K 影片，不会卡顿。 |
| **网络速度** | **True 千兆以太网**。 | 保证从局域网 NAS 或电脑上串流高码率 4K 影片时不缓冲。 |
| **USB 3.0** | 两个高速 USB 3.0 接口。 | 外接高速 SSD 或移动硬盘播放本地文件时速度更快。 |
| **RAM** | 4GB/8GB 可选。 | 确保 Kodi 运行时流畅，处理大型媒体库和海报信息时内存足够。 |

### 3. 费用节约小贴士

* **内存选择：** 如果预算紧张，**4GB** 版本绝对够用。
* **电源：** 确保使用手头质量好的 **5V/3A 或 5.1V/3A** 手机充电头（Type-C 接口）即可，不必追求官方电源。
* **外壳：** 可选最基础的散热片+透明外壳，费用更低。

**总结：**

在不计入外置硬盘成本的情况下，一个稳定、高性能的树莓派 4B 高清媒体播放器，核心硬件的投入大约在 **600 元至 1200 元**之间，具体取决于内存大小和外设的品牌选择。这个成本远低于市面上的高端媒体播放盒子，且具有极高的可定制性和扩展性！
